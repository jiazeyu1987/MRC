# Task 1: LLM调试面板 - 简单实现

## 需求概述 (来自 info.txt)

1. **在【会话剧场】里右侧添加面板【LLM调试面板】，常显示**
2. **在调用LLM的函数里增加触发器，当使用LLM给提示词的时候，收到返回消息的时候，提示词和返回消息在LLM调试面板里显示**
3. **这是一个简单的任务，用尽量简单的方式实现，不用考虑数据安全，错误处理**

## 核心原则

✅ **简单至上** - 代码量控制在100行以内
✅ **直接实现** - 避免过度架构和复杂设计
✅ **快速完成** - 1-2小时内可以完成
✅ **最小改动** - 尽量少修改现有代码

## 技术方案

### 采用最简单的方案：API响应增强

**数据流**:
```
用户点击"下一步" → 后端LLM调用 → API响应包含debug信息 → 前端显示到调试面板
```

**优势**:
- 不需要WebSocket或实时通信
- 不需要复杂的状态管理
- 代码修改量最小
- 完全符合"简单实现"要求
